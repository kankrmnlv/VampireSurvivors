1. Инструкция по запуску:
-Распаковать архив, запустить My project.exe
-Игра начнется автоматически
-Alt+F4 для закрытия игры

2. Архитектура:
Features/Data:
-Дата слой: скрипты DTO - держатели данных, которые берутся из balance.json
-Синглтон BalanceManager для получение всех балансных данных
Features/Player:
-Player скрипты, для передвижение, управление здоровьем
Features/Enemy: 
-Enemy скрипты, для передвижения, стремления к игроку, спавнер врагов
Features/Combat:
-Скрипты для взаимодействия между инстансами, нанесение и получение урона, логика ауры и снаряда
Features/LevelXp и Features/Skills:
-Скрипты по логике опыта, уровня, повышения уровня и апгрейд навыков
Features/UI:
-Скрипты по интерфейсам
Features/Pool:
-Логика пул системы

3. Использование AI:
Был использован ChatGPT 5.2, и использованием только чата - отправление промптов.
Сгенерированы были абсолютно все скрипты, с последующим внесением правок по код стайлу, оптимизацией логики, перехода с Update на корутинный и event Action подход.
Почти не было внесено правок с моей стороны в логику только пул-системы и Feauture/Skills системы.

Были промпты только по коду, сетап сцены и работа с компонентами была сделана без использования AI.

Что мог сделать лучше:
1) Работа с UI не очень красивая, хард код на 3 скилла, прямые ссылки на кнопки и текста. Из-за временных рамок оставил как есть.
2) Пул врагов - не успел оптимизировать чтоб вместо Instantiate/Destroy враги тоже работали на пул системе.
3) Player - не через префаб, а инстанс на сцене, использующий прямые ссылки на некоторые компоненты. По хорошему нужно было не так сильно связывать игрока к сцене, но т.к. тут такой маленький MVP геймплей, не стал усложнять чтоб успеть по временным рамкам
